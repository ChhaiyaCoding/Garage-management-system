# Telegram Bot · ការ​ដំឡើង

​មុខ​ងារ Telegram Bot នៅ​ Garage OS អនុញ្ញាត​ឱ្យ:

- 📩 **ផ្ញើ​សារ​ទៅ​អតិថិជន** ​នៅ​ពេល​ Job ​ផ្លាស់​ប្ដូរ​ស្ថានភាព (ភ្ជាប់​តាម Bot ​ឬ​តាម​ត្រូវ​ដៃ​ផ្ទាល់)
- 🔔 **ជូន​ដំណឹង​ដល់​ម្ចាស់​ហាង** ​នៅ​ពេល​មាន booking ​ថ្មី ឬ Parts ​ស្ទើរ​អស់
- 🔗 **ចែករំលែក Invoice / Quote** ​តាម Telegram (one-click share URL)

---

## ​ជំហាន​ដំឡើង

### 1️⃣ ​បង្កើត Telegram Bot

1. ​បើក Telegram → ស្វែងរក **`@BotFather`**
2. ​ផ្ញើ **`/newbot`**
3. BotFather ​នឹង​សួរ:
   - **Bot name** — ​ដាក់​ឈ្មោះ​ឱ្យ bot របស់​អ្នក (ឧ. "Garage Sok Pheap")
   - **Username** — ​ត្រូវ​បញ្ចប់​ដោយ `_bot` (ឧ. `sokpheap_garage_bot`)
4. បន្ទាប់​មក BotFather ​នឹង​ផ្ដល់ **Bot Token** ​ឱ្យ​អ្នក​មួយ​ដែល​មាន​ទ្រង់ទ្រាយ:
   ```
   123456:ABC-DEF1234567890_ABCDEFGHIJKLMNOPQR
   ```
   ​សូម​រក្សា​វា​ឱ្យ​សម្ងាត់!

### 2️⃣ ​យក Chat ID ​របស់​អ្នក

1. ​នៅ Telegram → ស្វែងរក **`@userinfobot`**
2. ​ផ្ញើ **`/start`**
3. Bot នឹង​ឆ្លើយ​ឱ្យ​អ្នក​មាន​ដូច​នេះ:
   ```
   ID: 123456789
   First: Sok
   ```
4. Copy **ID** ​នោះ — ​នេះ​ជា **Owner Chat ID** ​របស់​អ្នក

### 3️⃣ ​ផ្ញើ /start ទៅ Bot ​ដែល​អ្នក​បាន​បង្កើត

**សំខាន់​ខ្លាំង!** Bot មិន​អាច​ផ្ញើ​សារ​ដំបូង​ឱ្យ​អ្នក​បាន​ទេ — ​អ្នក​ត្រូវ​ផ្ញើ​មុន។

1. បើក Telegram → ស្វែងរក​ bot ​ដែល​អ្នក​បាន​បង្កើត (តាម username ឧ. `@sokpheap_garage_bot`)
2. ​ចុច **Start** ឬ​ផ្ញើ **`/start`**

### 4️⃣ ​ភ្ជាប់​ទៅ Garage OS

1. ​បើក Garage OS → ​ចូល **Settings** (​ការ​កំណត់)
2. ​ចុច​ tab **Integrations**
3. ​នៅ​ Card "Telegram" → ​ចុច **ភ្ជាប់**
4. បំពេញ:
   - **Bot Token**: token ​ដែល BotFather ផ្ដល់
   - **Chat ID ​របស់​ម្ចាស់​ហាង**: ID ​ដែល @userinfobot ​ផ្ដល់
5. ជ្រើស​ ការ​ជូន​ដំណឹង​ស្វ័យ​ប្រវត្តិ:
   - ✅ មាន booking ​ថ្មី
   - ✅ Parts ស្ទើរ​អស់
6. ​ចុច **សាក & រក្សា​ទុក**

​ប្រសិន​បើ​ត្រឹមត្រូវ:
- ​បាន​ភ្ជាប់ → bot ​នឹង​ផ្ញើ​សារ​សាក​ឱ្យ​អ្នក​ភ្លាមៗ ✅
- ​បាន​បរាជ័យ → ​ឆែក token ឬ chat ID

## ​ការ​ប្រើ​ប្រាស់

### ​ផ្ញើ​សារ​ទៅ​អតិថិជន

​ក្នុង **JobDrawer** ​ចុច​ប៊ូតុង **"ផ្ញើ Telegram"**:

- ​បើ​ Bot បាន​ភ្ជាប់ + ​អតិថិជន​មាន `telegramChatId` → ​សារ​ត្រូវ​បាន​ផ្ញើ​ដោយ​ស្វ័យ​ប្រវត្តិ
- ​បើ​មិន​មាន chat ID → ​បើក t.me/share/url ​ដែល​ឱ្យ​ម្ចាស់​ហាង​ផ្ញើ​ដោយ​ខ្លួន​ឯង (ផ្ទេរ​សារ​ទៅ chat ​អតិថិជន​ផ្ទាល់)

### ​ចែករំលែក Invoice/Quote

- ​បើក Invoice ​ឬ Quote modal → ​ចុច **"ផ្ញើ​តាម Telegram"**
- ​នឹង​បើក Telegram ​ដោយ​មាន​សារ​រួច​ចួន​ហើយ​ → ​ម្ចាស់​ហាង​ជ្រើស chat ​ដែល​ត្រូវ​ផ្ញើ

### ​ជូន​ដំណឹង​ស្វ័យ​ប្រវត្តិ

- **Booking ថ្មី**: ​នៅ​ពេល​មាន booking ​ថ្មី​ត្រូវ​បាន​បង្កើត → ​ម្ចាស់​ហាង​ទទួល​សារ​ភ្លាមៗ
- **Low stock**: ​នៅ​ពេល part ​មួយ​ឆ្លង​ពី​ "មាន​គ្រប់​គ្រាន់" → "ស្ទើរ​អស់" (`stock <= reorder`) → ​ម្ចាស់​ហាង​ទទួល​សារ​អំពី​ SKU, ​បរិមាណ​នៅ​សល់, ​និង​កម្រិត reorder។ ​ផ្ញើ​មាន​តែ​ម្ដង​ប៉ុណ្ណោះ​ក្នុង​មួយ​ session​ (មិន spam) — ​ពេល​ restock ​លើស threshold ​ហើយ​ធ្លាក់​ម្ដង​ទៀត​ទើប alert ​ឡើង​វិញ។

## ​សុវត្ថិភាព

- Bot Token ​ត្រូវ​បាន​រក្សា​ទុក​ក្នុង Supabase JSON blob (workspace) ​របស់​ user ដែល​ sign in
- Token ​មិន​ត្រូវ​បាន​ចែករំលែក​ឱ្យ​អ្នក​ដទៃ​ឡើយ
- Telegram API ​ត្រូវ​បាន​ហៅ​ដោយ​ផ្ទាល់​ពី browser → no backend ​ត្រូវ​ការ
